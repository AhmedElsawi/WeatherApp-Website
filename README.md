# Weather Forecast Dashboard

A simple and responsive weather application that provides real-time weather information for any city worldwide. Built to practice API integration, frontend development, and clean UI design.

---

## Features

- Search weather by city name
- Real-time temperature, humidity, and wind speed
- Dynamic weather conditions (clear, cloudy, rain, etc.)
- Responsive UI for mobile and desktop
- Error handling for invalid city searches

---

## Key Learnings

This project helped reinforce:
- Working with REST APIs
- Asynchronous JavaScript (fetch/async-await)
- DOM manipulation
- UI/UX design basics
  
---

## Tech Stack

- HTML5
- CSS3
- JavaScript 
- OpenWeatherMap API 

---

## API Used

This app fetches weather data from a third-party weather API.
- User enters city name
- App sends request to weather API
- Response is parsed and displayed dynamically
